Usage
=====

Installation
------------

To use nlpertools, first install it using pip:

```console
(.venv) $ pip install nlpertools
```

Writing a file
----------------

To write a txt file, you can use the
`nlpertools.readtxt_list_all_strip()` function:

::: src.nlpertools.readtxt_list_all_strip
